# Политика безопасности

## Сообщение об уязвимостях

Если вы обнаружили уязвимость в модуле, пожалуйста, не создавайте публичный issue.
Свяжитесь с автором напрямую через сайт [https://bazarow.ru](https://bazarow.ru).

## Поддерживаемые версии

| Версия | Поддержка |
|--------|-----------|
| 1.0.x | Активная поддержка |

## Меры безопасности модуля

- Python-сервис слушает только `127.0.0.1:9876` (недоступен извне)
- AJAX-контроллер требует CSRF-токен
- Все входные данные экранируются
- Парольные данные не хранятся в открытом виде
- Прямой доступ к Python-скриптам из веба запрещён через `.htaccess`

## Обновления

Обновления модуля публикуются через [https://bazarow.ru](https://bazarow.ru).
